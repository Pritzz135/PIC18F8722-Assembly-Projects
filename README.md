
# PIC18F8722 Assembly Programs
**Name:** Prithvi Snehal  
**Student ID:** 11691120  

A collection of PIC18F8722 assembly programs written in MPLAB X 
using XC8 assembler, as part of my microcontroller coursework.

---

## Hardware
- Microcontroller: PIC18F8722
- IDE: MPLAB X
- Assembler: XC8
- Peripherals used: PORTF (LEDs), PORTJ (pushbuttons), 
  PORTH (switches), PORTC, PORTA (enable/transistor)

---

## Notes
- Delay subroutines are calibrated for the lab board clock speed
- Each file has its own self-contained delay subroutines
